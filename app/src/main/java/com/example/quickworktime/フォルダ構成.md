技術スタック

UI: Data Binding, ViewBinding, ConstraintLayout
非同期処理: Coroutines + Flow
データベース: Room ORM
アーキテクチャ: MVVM + Repository Pattern
DI: 手動依存性注入（Dagger未使用）
